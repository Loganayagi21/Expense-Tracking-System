# Expense Tracking System

An Expense Tracking System is a web-based application designed to help users record, manage, and analyze their daily expenses efficiently.  
This project allows users to add, update, delete, and visualize expenses by category and month.

---

## Features

- Add daily expenses
- Update existing expense records
- Delete expenses
- View expense history
- Monthly expense analytics
- Category-wise expense breakdown
- Interactive charts and graphs
- API integration using FastAPI
- User-friendly frontend using Streamlit
- Database storage using MySQL

---
